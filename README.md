# Investment Mindset Training Agent

A personal investment simulation agent that helps you visualize asset allocation, track gains/losses, and calculate your path to financial freedom.

## Features

- **Portfolio Dashboard** - Track all your investments across multiple markets (US stocks, A-shares, crypto) with real-time prices
- **Financial Freedom Timeline** - Set goals and see how long until your passive income covers your expenses
- **Investment Journal** - Record the reasoning behind every trade for future review
- **Asset News** - Stay updated with news related to your holdings
- **Email Reports** - Generate and send investment summary reports

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React + Vite + TypeScript |
| Backend | FastAPI + Python |
| Database | SQLite |
| Data Sources | yfinance + akshare |
